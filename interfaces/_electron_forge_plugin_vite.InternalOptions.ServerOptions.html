Optional proxy

proxy?: Record<string, string | ProxyOptions>

Inherited from CommonServerOptions.proxy

  • Defined in node_modules/vite/dist/node/index.d.ts:358

Configure custom proxy rules for the dev server. Expects an object of { key: options } pairs. Uses http-proxy. Full options here.

Example vite.config.js:

module.exports = {
  proxy: {
    // string shorthand
    '/foo': 'http://localhost:4567/foo',
    // with options
    '/api': {
      target: 'http://jsonplaceholder.typicode.com',
      changeOrigin: true,
      rewrite: path => path.replace(/^\/api/, '')
    }
  }
}
